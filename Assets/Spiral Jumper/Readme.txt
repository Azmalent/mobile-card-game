Для корректной работы терубется объявить несколько новых слоев:
1) Коллайдер персонажа должен находиться на слое «Ignore Raycast» или на любом другом, но тогда нужно модифицировать все рэйкасты, связанные с игроком, чтобы они игнорировали этот слой.
2) 9 - «PlayerCenterPoint». Используется для коллайдера центра персонажа. Не пересекается ни с какими слоями, даже с самим собой.
3) 10 - «Platform Effector». Используется для коллайдеров активной платформы.  Пересекается только со слоем коллайдера персонажа. 
4) 11 - «Amorph Physics» Используется для коллайдеров, задействованных в аморфной физике. Не пересекается ни с какими слоями, даже с самим собой. Расположен в 2D.
